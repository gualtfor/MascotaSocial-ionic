# MascotaSocial 🐾

MascotaSocial es una red social móvil dedicada completamente a las mascotas. Permite a los usuarios compartir momentos, ver videos entretenidos y conectar con otros amantes de los animales en todo el mundo.

## 🚀 Características

*   **Inicio (Feed):** Ve las últimas actualizaciones, historias y publicaciones de la comunidad.
*   **Galería de Fotos:** Un hermoso diseño de cuadrícula con filtros por categoría (Perros, Gatos, Aves, Exóticos).
*   **Reels de Video:** Un feed de videos inmersivo con desplazamiento vertical.
*   **Perfiles de Usuario:** Administra el perfil de tu mascota, revisa tus estadísticas de seguidores y edita fácilmente tus datos de contacto y ubicación.

## 🛠️ Tecnologías

*   **Framework:** [Ionic Framework](https://ionicframework.com/)
*   **Lógica e Interfaz:** [Angular](https://angular.io/)
*   **Puente Nativo:** [Capacitor](https://capacitorjs.com/) (para compilación en Android)
*   **Estilos:** SCSS y Variables CSS
